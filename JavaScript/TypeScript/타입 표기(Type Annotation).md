- 타입스크립트에서는 이렇게 **`:`** 를 이용하여 타입을 정의하는데, 이를 타입 표기(Type Annotation) 라고 한다.

- 기존 자바스크립트에서 쓰이던 타입들도 있는 반면 튜플, 이넘 등 다른 언어에서는 흔히 쓰이지만 자바스크립트에서는 볼 수 없었던 타입들도 등장한다.
