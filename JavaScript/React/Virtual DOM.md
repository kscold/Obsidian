- DOM를 추상화한 자바스크립트 객체이며, 실제 DOM의 사본에 가깝다.


### 리액트에서 동작과정
1. 데이터를 업데이트하면 전체 UI를 Virtual DOM에 리렌더링한다.
2. 이전 Virtual DOM에 있던 내용과 현재 내용을 비교한다.
3. 바뀐 부분만 실제 DOM에 적용한다.
