


## primitive type의 종류

- short, int, long, float, double, char, boolean, byte 등이 있다.



## reference type의 종류

- 참조 자료형은 위의 기본 자료형을 제외한 모든 자료형을 말한다. 

- 즉, 자바의 [[인스턴스(Instance)]]를 가리킬 수 있는 자료형이다. 
- [[클래스(Class)]], [[배열(Array)]], [[열거(Enum)]], [[인터페이스(Interface)]] 등이 존재한다. 
- String 또한 참조 자료형이다.
