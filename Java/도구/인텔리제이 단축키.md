- IntelliJ IDEA는 JetBrains에서 만든 Java/Kotlin 개발에 최적화된 IDE이다.
- macOS 기준 단축키이며, Windows에서는 `command` → `Ctrl`, `option` → `Alt`로 대응된다.

## 실행 및 디버그

| 단축키 | 기능 |
| ---- | ---- |
| `ctrl + R` | 이전에 선택했던 메서드/클래스 실행 |
| `ctrl + shift + R` | 현재 파일의 메서드/클래스 실행 |
| `command + F2` | 컴파일(프로세스) 종료 |
| `command + shift + F9` | 리컴파일 및 재실행 |
| `F2` | 가장 가까운 오류 위치로 이동 |

## 코드 편집

| 단축키 | 기능 |
| ---- | ---- |
| `command + D` | 현재 줄 복사 후 바로 아래에 삽입 |
| `command + /` | 한 줄 주석 `//` 토글 |
| `command + shift + /` | 블록 주석 `/* */` 토글 |
| `command + shift + Enter` | 자동 세미콜론(`;`) 추가 후 커서 이동 |
| `shift + F6` | 현재 심볼(변수/메서드명) 전체 rename |
| `option + 위/아래` | 코드 블록 영역 확장/축소 선택 |

## 리팩토링

| 단축키 | 기능 |
| ---- | ---- |
| `command + option + M` | 선택 영역을 메서드로 추출 |
| `command + option + V` | 선택 영역을 변수로 추출 |
| `command + option + L` | 선택 영역 코드 포맷팅 |

## 탐색

| 단축키 | 기능 |
| ---- | ---- |
| `command + E` | 최근 이동한 파일 목록 표시 |
| `command + 정의 클릭` | 정의된 코드 위치로 이동 |
| `ctrl + G` | 같은 단어를 누른 횟수만큼 다중 선택 |

## 자동완성 및 생성

| 단축키 | 기능 |
| ---- | ---- |
| `command + N` | 생성자/getter/setter 등 코드 생성 다이얼로그 |
| `command + P` | 자동완성 파라미터 힌트 (VS Code의 `command + I`와 유사) |
| `option + Enter` | 미설치 라이브러리 다운로드 / import / 빠른 수정 목록 |

## 창 관리

| 단축키 | 기능 |
| ---- | ---- |
| `command + W` | 현재 편집 창 닫기 |

## 자주 쓰는 라이브 템플릿

| 입력 | 결과 |
| ---- | ---- |
| `psvm` | `public static void main(String[] args) {}` 생성 |
| `sout` | `System.out.println("")` 생성 |
| `soutv` | `System.out.println("변수명 = " + 변수명)` 자동 매핑 생성 |
