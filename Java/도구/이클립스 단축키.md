- 이클립스(Eclipse)는 Java 개발에 널리 쓰이는 오픈소스 IDE이다.
- 단축키를 익히면 생산성이 크게 향상된다.
- macOS 기준 단축키이며, Windows에서는 `command` → `Ctrl`, `option` → `Alt`로 대응된다.

## 코드 편집

| 단축키 | 기능 |
| ---- | ---- |
| `option + command + S` | getter/setter 등 코드 스니펫 생성 |
| `control + space` | 자동완성 및 추천 메서드 목록 표시 |
| `command + shift + F` | 코드 자동 포맷팅 |
| `command + shift + O` | 필요한 라이브러리 자동 import |

## 파일 탐색

| 단축키 | 기능 |
| ---- | ---- |
| `command + N` | 새 파일(Java 클래스 등) 생성 |

## 자주 쓰는 라이브 템플릿

| 입력 | 결과 |
| ---- | ---- |
| `sysout` + `control + space` | `System.out.println()` 자동완성 |
