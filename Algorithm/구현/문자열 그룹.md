- 연속한 문자열은 가능하나, 떨어진 일전에 나왔던 문자는 버리는 구현이다.

```python

n = int(input())  
strings = []  
for _ in range(n):  
    strings.append(input())  
  
count = 0  
  
for string in strings:  
    char_set = set()
    is_group_word = True  
  
    for char in string:  
        if char in char_set and char != prev_char: 
        # 이전 문자와 현재 문자가 같지않는데, 나온적이 있는 경우
            is_group_word = False  
            break  
        char_set.add(char)  
        prev_char = char  
  
    if is_group_word:  
        count += 1  
  
print(count)
```
